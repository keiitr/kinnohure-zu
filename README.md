# 銀フレ 001–100 単語帳 PWA

TOEIC Score 600レベルの単語 001〜100 を、フレーズ穴埋め形式で覚えるための単語帳アプリです。
iPhone / iPad のホーム画面に追加して、オフラインでも使えます(英語音声つき)。

## ファイル構成

```
index.html      … アプリ本体(単語データ・画面・ロジックすべて入り)
manifest.json   … PWA設定(アプリ名・アイコン)
sw.js           … Service Worker(オフラインキャッシュ)
icon-180.png    … ホーム画面アイコン(iOS用)
icon-192.png    … アイコン(Android・PWA用)
icon-512.png    … アイコン(大)
```

**この6ファイルすべてを、同じ階層に置いてください。**

## GitHub Pages へのデプロイ手順

1. GitHubで新しいリポジトリを作る(例: `ginfure-100`)。**必ず Public** にする(無料プランではPrivateリポジトリのPagesは使えません)
2. 上の6ファイルを **リポジトリの一番上(ルート)** にアップロードする
   - 「Add file → Upload files」で6ファイルを直接ドラッグ
   - ⚠️ `ginfure-100` などのフォルダごとドラッグしない(1階層深くなりURLが変わります)
3. リポジトリの **Settings → Pages** を開く
4. 「Build and deployment」の Source を **Deploy from a branch** にする
5. Branch を **main**、フォルダを **/(root)** にして **Save**
6. 1〜3分待つと、ページ上部に URL が表示される:
   ```
   https://ユーザー名.github.io/リポジトリ名/
   ```
7. その URL を iPhone の Safari で開く → 共有ボタン → **「ホーム画面に追加」**

## 404 Not Found になるときのチェックリスト

上から順に確認してください。ほとんどはこのどれかです。

1. **Pagesが有効になっていない**
   Settings → Pages で Source が「None」のままになっていないか。手順4〜5を実行する。

2. **URLの打ち間違い**
   正しい形は `https://ユーザー名.github.io/リポジトリ名/` です。
   - 末尾に `/index.html` は不要(あっても動きますが)
   - `github.com/…` ではなく `github.io` です
   - ユーザー名・リポジトリ名の大文字小文字も一致させる

3. **ファイルが1階層深い**
   リポジトリを開いて、トップに `index.html` が直接見えるか確認。
   `tango/index.html` のようにフォルダの中にある場合は、
   - ファイルをルートに移動する、または
   - URLを `https://ユーザー名.github.io/リポジトリ名/tango/` にする

4. **反映待ち**
   Pages有効化やファイル更新の直後は、反映まで数分かかります。
   リポジトリの **Actions** タブで「pages build and deployment」が緑のチェックになってから開く。

5. **リポジトリがPrivate**
   無料プランではPublicのみPages対象。Settings → General の一番下 → Change visibility で Public に変更。

6. **ブランチ名がmainでない**
   古い設定だと `master` の場合があります。Pages設定のBranchを実際のブランチ名に合わせる。

7. **キャッシュが古い(更新が反映されない場合)**
   404ではなく「更新したのに古いまま」のときは、Service Workerのキャッシュが原因です。
   `sw.js` の1行目 `ginfure-v2` の数字を上げてコミットすると、次回起動時に新版へ更新されます。
   手元のSafariでは、設定 → Safari → 詳細 → Webサイトデータ から該当サイトを削除でも可。

## 使い方

- **カードをタップ** … 答え(単語・意味)を表示。「🔊 自動」ONなら同時に英語音声を再生
- **まだ / 覚えた** … 自己判定して次へ。進捗は端末に自動保存
- **チップ(001–010など)** … 出題範囲の切り替え。全問「覚えた」になるとオレンジの点が付く
- **🔀 シャッフル** … 出題順をランダムに
- **苦手のみ** … 「覚えた」以外だけを周回。覚えたカードはその場でデッキから抜ける
- **🔊 自動** … 答え表示時の自動読み上げのON/OFF
- **一覧** … 全単語リスト。行をタップで「覚えた」チェックの付け外し
- **カード右下の🔊** … いつでも手動再生(答え表示前はフレーズ全文、表示後は単語→フレーズ)

## 補足

- 進捗データはブラウザのlocalStorageに保存されます。Safariのサイトデータを削除するとリセットされます
- iPhoneのマナーモード(サイレントスイッチON)では音声が鳴らない場合があります
- 音声はiOS内蔵の音声合成(Web Speech API)を使用。通信・APIキーは不要です
